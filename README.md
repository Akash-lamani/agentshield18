# AgentShield Dashboard — AI Security Platform

## Quick Start
```bash
cd agentshield-dashboard
npm install
npm run dev       # http://localhost:5173
npm run build     # Production build
```

## Hackathon Demo Flow
1. Dashboard → show score 28/100
2. Vulnerabilities → filter by critical, expand rows
3. Watch Mode → start watching
4. Attack Simulator → inject attacks (WOW moment!)
5. Apply Fix → Rescan → score jumps to 82/100
6. Reports → download JSON/HTML
7. Terminal → type 'help'
